## ⭐️ 소개

- 커스텀이 자유로운 블로그에 대한 관심으로 진행한 사이드 프로젝트입니다.
- 기능 구현보다 학습을 우선으로 스타터 없이 제작했습니다.
- 사이드 프로젝트 개념으로 필요한 기능들을 지속적으로 구현하고 있습니다.
